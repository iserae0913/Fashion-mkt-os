# THESIS｜Chic 通勤

## 1. 一句话定义
- Chic 通勤不是传统职业装，而是一种把都市女性的效率感、体面感、聪明感与可穿性结合起来的长期风格命题。

## 2. 这是什么类型的命题
- [x] 审美母题
- [x] 身份脚本
- [x] 场景母题
- [ ] 标志性单品
- [ ] 语义词库
- [ ] 平台心智赛道

## 3. 核心情绪
- 体面但不刻板
- 质感但不张扬
- 有能力感但不强攻击性
- 有上升感和自我价值确认

## 4. 它回应了什么消费者心理
- 都市女性希望通过穿搭呈现更稳定、更高级、更被认可的自己。
- 用户不只想买一件衣服，而是想买一种“我已经进入更好状态”的感觉。
- 它同时回应了效率、身份、审美和情绪价值。

## 5. 海外源头
- 来源市场：韩国 / 日本 / 全球都市女性时尚内容
- 来源人物/品牌/平台：MUSINSA、ZOZOTOWN、office chic 内容生态
- 最初表达方式：极简通勤、高级日常、都市轻权力感
- 海外语义：office chic / smart commute / refined everyday dressing

## 6. 中国翻译层
- 中文关键词：Chic 通勤、高智感通勤、通勤升级、都市贵气感
- 小红书表达：通勤 OOTD、早八穿搭、办公室也要有气场、聪明感穿搭
- 抖音表达：上班变装、显贵通勤、气质通勤、办公室高级感
- 得物/交易表达：包袋、鞋履、轻潮流通勤单品
- 是否形成稳定中文命名：正在形成中，值得持续跟踪

## 7. 对应身份脚本
- 谁会拥抱这个命题：一二线城市白领、内容敏感用户、希望显得更专业又更有气质的女性
- 典型人设：早八女生、都市白领、脑力型女性、轻中产审美用户
- 社会情绪 / 欲望结构：上升、可控、体面、自我确认

## 8. 对应场景
- 核心场景：上班通勤、办公室、客户会面、咖啡办公
- 次级场景：下班后社交、短途差旅、城市移动
- 是否跨场景：是
- 是否跨季节：是

## 11. 平台表现

- 搜索性：高潜力

- 用户讨论点：显贵、聪明感、上班要体面、低调高级

### 抖音
